# ClubOps Dashboard - Project Specification

## Project Name

ClubOps Dashboard

## Description

ClubOps Dashboard is a football club operations management dashboard built with React, TypeScript, Java Spring Boot, and MySQL.

It allows users to manage a simulated football club's squads, players, staff, contracts, finances, facilities, and operational analytics.

This is not a football game or match simulation project. It is a full-stack club management system focused on relational database design, REST APIs, authentication, CRUD operations, SQL aggregation, and dashboard analytics.

- User registration and login

- Club profile
- Teams: First Team, B Team, U19
- Player management

- Staff management
- Contract management
- Finance page
- Facilities page

## Non-Goals

This project will not include:

- Match simulation
- Real player data
- Real club data
- Transfer market simulation
- Tactical board
- Training system
- League standings
- Player growth algorithm
